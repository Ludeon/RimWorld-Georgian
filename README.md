RimWorld-Georgian

Official Georgian language translation for RimWorld.

This repository contains the Georgian localization files for RimWorld and its DLCs, maintained in the modern multi-DLC repository layout.

Repository structure

Core/

Royalty/

Ideology/

Biotech/

Anomaly/

Odyssey/

Each folder contains the Georgian language files for the corresponding game module.

Status

The Georgian translation is currently in active development.

Main focus areas:

translation QA and polishing

fixing leftover English text

shortening UI strings where needed

improving terminology consistency

testing in-game for readability and layout issues

Contributing

If you would like to help improve the Georgian translation:

report untranslated or incorrect strings

report UI text overflow issues

suggest terminology improvements

submit pull requests with focused changes

Please preserve:

XML structure

keys and tag names

placeholders such as {0}, {1}, {PAWN_nameDef}

escape sequences such as \n

Notes

This repository is structured in the same modern layout used by other actively maintained RimWorld translation repositories.

Further translation cleanup and DLC polishing will be submitted in follow-up updates.
